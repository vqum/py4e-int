8.11. Multiple Choice Questions¶

    Q-1: Which of the following commands is used to open a file called myText.txt in Read-Only mode?

  • infile = open(myText.txt, “r”)
  • Using the file name without quotation marks makes Python think the file name is a variable name here.
  • infile = open("myText.txt", “r”)
  • We provide a string with file name + "r" which means read only.
  • infile = open("myText.txt", “w”)
  • "w" opens the file in writing mode.

    Q-2: Which of the following commands is used to open a file called myText.txt in Write-Only mode?

  • outfile = open("myText.txt", "w")
  • We provide a string with file name + "w" which means write only.
  • outfile = open("myText.txt", “r”)
  • "r" opens the file in reading mode.
  • outfile = open(myText.txt, “w”)
  • Using the file name without quotation marks makes Python think the file name is a variable name here

    Q-3: Which command below closes the already open file myText.txt if the following code has already been written?

    ref_file = open("myText.txt", "r")
  • "myText".close()
  • The title of the file is a string and a string cannot be closed in Python.
  • ref_file.close()
  • This closes the file using the variable it has been assigned to.
  • close(ref_file)
  • The command close() needs to be called on the file's variable using dot notation.
  • close("myText")
  • The command close() needs to be called on the file's variable using dot notation.

    Q-4: Which of the commands below is used to add the following string to the end of a file referenced with the variable filevar?

    somestring = "my Sentence"
  • filevar.append(somestring)
  • Append is a command used for lists, not files.
  • filevar.write("somestring")
  • somestring is a variable and does not need quotation marks around it.
  • filevar.write(somestring)
  • Using dot notation, we can call the write command with the string variable inside the parentheses.
  • somestring.write()
  • The command write() needs to be called on the file's variable, not the string itself.

    Q-5: The contents of names.txt is listed here:

    Moana
Cinderella
Tiana

    Which of the following code blocks will print all of the names in names.txt?

    I

    names = open("names.txt", "r")
for line in names:
    print(names)

    II

    names = open("names.txt", "r")
for line in names:
    print(line)

    III

    names = open("names.txt", "r")
for line in names:
    print("line")
  • I
  • Append is a command used for lists, not files.
  • II
  • somestring is a variable and does not need quotation marks around it.
  • III
  • Using dot notation, we can call the write command with the string variable inside the parentheses.
  • None of the above.

    Q-6: What type of loop was used in this chapter to iterate through the lines in a file?

  • For loop
  • A for loop is typically used to iterate through the lines in a file.
  • While loop
  • A while loop can be used to run through lines in a file, but to iterate through, it is more common to use a for loop.
